# 模型主要思路
## &emsp; 模拟画家对于素描的绘制过程，将传统的人脸到素描之间一对一映射，变成了先描绘出来五官轮廓，再逐渐增加更多的细节，最后绘制出形象的人像素描。难点：利用前一步图生成的信息，辅助后一步生成器合成
